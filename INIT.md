# Init Guide

Use this quick init sequence when resuming work in this repository.

## 1. Load context
- Read `SKILL.md`
- Read `PROJECT_MEMORY.md`
- Read `ROADMAP.md`

## 2. Start local preview
```powershell
python -m http.server 4180
```

## 3. Open app
```text
http://127.0.0.1:4180/
```

## 4. Validate before delivery
```powershell
node --check app.js
```
